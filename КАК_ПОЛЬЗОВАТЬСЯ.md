# generate.py — генератор setup-скриптов (как форма на сайте, но локально)

Один источник данных → собирает все 6 скриптов с подстановкой везде
(адреса, VLAN, NAT, GRE, OSPF, DHCP, DNS-зоны). Python есть на Альте всегда.

## Способ 1 — отредактировать конфиг (рекомендую)
1. Открой generate.py, найди блок `CONFIG = {` в начале.
2. Поменяй значения под своё задание (адреса сетей, VLAN, ключ, интерфейсы, TZ).
3. Запусти:
       python3 generate.py
4. Готовые скрипты появятся в папке out/. Запуск на каждой машине:
       sudo bash out/isp.sh      # ISP
       sudo bash out/hq-rtr.sh   # HQ-RTR   ... и т.д.

## Способ 2 — ввести значения по очереди
       python3 generate.py --ask
   (Enter = оставить значение из CONFIG)

## Что считается автоматически
- адреса ISP/роутеров из сети (.1 = ISP, .2 = роутер)
- адреса туннеля (.1 = HQ, .2 = BR)
- маска и диапазон DHCP из сети VLAN200
- имена обратных DNS-зон из адресов серверов/клиента

## Важно
- Интерфейсы по умолчанию ens19/ens20/ens21 — сверь `ip -br link`, поменяй в CONFIG.
- VLAN tag 100/200 для HQ-SRV и HQ-CLI ставится в Proxmox (Hardware → VLAN Tag).
- Порядок запуска: ISP → HQ-RTR → HQ-SRV → BR-RTR → BR-SRV → HQ-CLI.
- Скрипты пишутся с LF (Unix), на Альте запускаются без проблем.
